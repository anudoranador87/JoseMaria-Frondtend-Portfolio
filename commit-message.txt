refactor(cv): full editorial redesign — from functional to professional

The original index.html was a working CV but lacked visual hierarchy,
typographic structure, and the design language expected from a front-end
developer's portfolio. This commit documents the complete redesign.

--- WHAT CHANGED ---

Typography
- Added Playfair Display (headlines), Inter (UI), Fira Code (labels/meta)
- Previous version used a single generic font with no hierarchy
- Headlines now use font-size: clamp() for fluid scaling across viewports

Header
- Rebuilt as 3-layer structure: meta bar / masthead name / contact bar
- Previous version: flat flexbox with photo, name and button in one row
- Photo placeholder integrated with border-radius and box-shadow
- Contact bar now includes email (real mailto:), GitHub and LinkedIn

Section separators
- Replaced dark h2 background blocks (#2c3e50 full-width) with editorial
  rule lines (line · label · line) — visual rhythm without visual noise
- h2 now communicates hierarchy through typography, not background color

Color system
- Introduced CSS Custom Properties for full light/dark mode token system
- --primary, --dark, --bg-body, --bg-section, --bg-card, --text-main,
  --text-soft, --border, --why-tech-bg, --skills-bg, --edu-bg, --exp-bg
- Dark mode: body.dark-mode overrides all tokens in one block
- Previous version: colors were partially hardcoded across multiple rules

Projects section
- Cards redesigned with tech tag pills (.ptag) per project
- EquiShift promoted to flagship card spanning full grid width
- Browser bar dots (red/yellow/green) kept as personality detail

Dev Log 365 section (new)
- Added as visual closing block before footer
- Dark background (#2c3e50) creates intentional contrast break in the page
- Shows: 17 entries written, 30+ bugs documented, 4 real projects, 91% cert avg
- 4 real diary excerpts with dates and topic tags
- CTA linking to Mi-Camino-Web-365 GitHub repository
- Blockquote: "La lógica es mi superpoder. La sintaxis es solo la herramienta."

Light mode contrast fix
- --text-soft darkened from #555555 to #3a3a3a (was failing WCAG AA)
- --text-main darkened from #1a1a1a to #111111
- Body copy in colored sections switched from --text-soft to --text-main

Fixes included in this commit
- Removed Cloudflare email obfuscation (/cdn-cgi/l/email-protection)
- Removed injected cloudflare-static/email-decode.min.js script tag
- Added rel="noopener noreferrer" to all external links
- Added LinkedIn (https://www.linkedin.com/in/joseaparicio87/)
- Fixed truncated script block caused by Cloudflare script injection
- Dark mode icon now toggles between fa-moon and fa-sun correctly

--- WHY THIS MATTERS ---

A front-end developer's CV is itself a portfolio piece.
The previous version demonstrated that HTML and CSS work.
This version demonstrates that design decisions are intentional.
EOF
