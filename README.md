### 📋 Changelog

#### v2.0 — Editorial redesign *(March 2026)*

The original CV was functional. This version is intentional.

- **Typography** — Playfair Display (headlines), Inter (UI), Fira Code (labels). `clamp()` for fluid headline scaling
- **Header** — Rebuilt as 3-layer structure: meta bar · masthead · contact bar. Added LinkedIn, fixed Cloudflare email obfuscation
- **Section separators** — Replaced full-width dark `h2` blocks with editorial rule lines (line · label · line)
- **Color system** — Full CSS Custom Properties token system. Dark mode via single `.dark-mode` class. Light mode contrast fix: `--text-soft` #555 → #3a3a3a (WCAG AA)
- **Dev Log 365 section** — New dark closing block with 17 real diary entries, stats and GitHub CTA
- **EquiShift flagship card** — Promoted to full-width grid showcase
- **Fixes** — Removed Cloudflare script injection, fixed truncated JS block, `rel="noopener noreferrer"` on all external links

#### v1.3 — Playing Cards & Dark Mode *(February 2026)*

- Completed FreeCodeCamp Playing Cards — Flexbox, 3D transforms, `cubic-bezier` timing, CSS `filter` grayscale-to-color
- Implemented native dark mode: `classList.toggle('dark-mode')` + CSS Custom Properties token system
- First real DOM manipulation with JavaScript Event Listeners

#### v1.2 — Projects & Portfolio *(February 2026)*

- Implemented Personal Projects section with Meta Certification labs
- Fixed layout and styling for project cards section

#### v1.1 — Styling & Visual Refactoring *(January 2026)*

- Refactored `index.html`: cleaner markup, fixed typos, repaired broken links
- Improved text alignment, spacing and header contrast for readability

#### v1.0 — Initial Structure & Semantics *(January 2026)*

- Semantic HTML structure: `<header>`, `<section>`, `<footer>` — proper document architecture
- Nested lists for professional skills, Education section with B.Ed. and certifications
- Cleaned markup, improved `alt` texts for accessibility and SEO
